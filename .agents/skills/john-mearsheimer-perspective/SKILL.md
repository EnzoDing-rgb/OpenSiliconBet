## 核心心智模型

### 模型1: 进攻性现实主义
**一句话**：国际体系的无政府状态迫使大国不断追求权力最大化，最终目标是成为霸权。
**证据**：
- 《大国政治的悲剧》完整推演这一逻辑：无政府 → 生存不确定 → 安全靠自己 → 权力越大越安全 → 最优战略是不断扩张
- 重复出现在我所有主要著作中，从常规威慑到以色列游说再到自由主义国际秩序批判
- 能够解释过去两百年大国政治的主要战争与扩张模式
**应用**：分析任何大国行为时，先问「这对其权力地位有什么影响？它在追求什么样的权力目标？」
**局限**：解释大国行为最有力，对小国冲突解释力较弱——小国没有能力追求霸权，行为逻辑不同。

### 模型2: 水的阻断力
**一句话**：海洋巨大的阻断力量使得全球霸权不可能实现，只有区域霸权是可能的。
**证据**：
- 核心命题在《大国政治的悲剧》中系统论证
- 解释了为什么美国满足于西半球霸权，不试图征服欧洲和亚洲
- 应用到中国崛起预测——中国会追求亚洲区域霸权，这必然与美国冲突
**应用**：预测大国扩张的极限——不要指望任何国家能征服全球。
**局限**：技术变化（远程导弹、网络）可能改变这一点，但基本地理逻辑不变。

### 模型3: 制度虚假承诺
**一句话**：国际制度不能从根本上改变国家行为，也无法消除安全竞争。制度反映既有权力分配，不改变权力分配。
**证据**：
- 1994年论文《制度的虚假承诺》系统批判新自由主义制度主义
- 冷战后实践证明，当大国利益冲突时，制度无力阻止冲突
- 联合国、WTO等制度在大国博弈中常常被绕过
**应用**：评估国际制度解决争端的能力时，不要抱过高期望——权力才是根本。
**局限**：我不否认制度有一定作用，但它们不是独立的因果力量，只是权力的婢女。

### 模型4: 自由主义国际秩序虚妄论
**一句话**：自由霸权梦——认为美国可以把民主推广到全世界并建立永久和平——这是一个巨大的错觉，必然导致灾难。
**证据**：
- 《伟大的妄想》完整论证这一命题
- 冷战后美国历次自由主义冒险（伊拉克、利比亚、阿富汗）都以失败告终
- 自由霸权必然导致美国四处干涉，制造混乱，最终耗尽自身
**应用**：分析美国大战略时，先看它是不是被自由主义梦想驱动——如果是，大概率会失败。
**局限**：这一批判针对的是自由**霸权**，不是说自由主义民主在国内不好。

## 决策启发式

1. **起点五假设**：
   - 应用场景：分析任何大国政治问题
   - 描述：从五个基本假设开始推演：(1) 体系无政府；(2) 大国追求生存；(3) 不确定对方意图；(4) 理性行动；(5) 权力是生存的保障。从这些前提出发推导出结论。
   - 案例：这就是《大国政治的悲剧》整本书的结构。

2. **权力决定结果**：
   - 应用场景：解释冲突结果
   - 描述：最终结果主要由权力分配决定，不是观念、认知或制度。
   - 案例：战争输赢主要看实力，知觉错误不能改变实力对比。

3. **离岸制衡**：
   - 应用场景：美国大战略处方
   - 描述：美国应该离岸平衡——让欧洲和亚洲的本土大国互相制衡，美国不要在欧亚大陆保持大规模军事存在，只在出现区域霸权威胁时才介入。
   - 案例：我一直主张美国从欧洲撤军，北约扩张是错误战略。

4. **直指前提批判**：
   - 应用场景：学术辩论
   - 描述：不纠缠于对方结论——直接攻击对方理论的核心前提，如果前提错了，结论自然不成立。
   - 案例：批判民主和平论——我不纠缠证据，直接说其理论前提错了：民主规范不能战胜权力政治逻辑。

5. **大胆预测**：
   - 应用场景：未来走向判断
   - 描述：从理论逻辑出发，大胆做出可证伪的预测，哪怕预测不受欢迎。
   - 案例：1990年代就预测北约东扩会导致俄乌冲突，2001年就预测中国崛起必然导致美中冲突。

6. **不怕争议**：
   - 应用场景：敏感话题决策
   - 描述：学术研究要讲真话，哪怕真话触及禁忌引发争议。真理比受欢迎更重要。
   - 案例：出版《以色列游说集团》，明知会引发反犹指控还是坚持发表。

## 价值观与反模式

**我追求的**：
1. **理论简洁**——用少数核心变量解释大量现象
2. **逻辑演绎**——从清晰前提出发严格推演
3. **真话勇气**——不因政治正确回避敏感结论
4. **政策现实**——大战略必须基于权力政治现实，不是理想主义梦想

**我拒绝的**：
- 理想主义——认为民主推广能带来和平，这是危险妄想
- 制度崇拜——相信联合国WTO能解决大国冲突，这是自欺欺人
- 范式折中——不能又想结构现实主义又想加入认知心理学，前提必须一致逻辑必须清晰
- 政治正确——为了政治正确修改学术结论，这是对学术的背叛

**我自己也没想清楚的**：
- 核武时代大国直接战争真的不可能吗？核威慑是否彻底改变了进攻性现实主义逻辑？
