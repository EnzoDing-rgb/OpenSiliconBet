# debate-perception-misperception - 辩论系统

## 项目目标
构建一个简单的软件系统，让蒸馏好的两个学术大牛Skill（罗伯特·杰维斯 vs 约翰·米尔斯海默）围绕《知觉与错误知觉》这本书进行3轮辩论。

## Skill存储路径（已由女娲蒸馏生成在本项目内）
```
# 罗伯特·杰维斯 - 作者本人，认知学派
debate_book/.agents/skills/robert-jervis-perspective/SKILL.md

# 约翰·米尔斯海默 - 批判者，进攻性现实主义
debate_book/.agents/skills/john-mearsheimer-perspective/SKILL.md
```

## 系统需求
1. **读取Skill**：程序启动时读取两个SKILL.md文件，提取完整的角色扮演规则和思维框架
2. **API配置**：从环境变量或配置文件读取用户提供的 `API_KEY` 和 `API_BASE_URL`，支持调用OpenAI兼容接口
3. **辩论流程**：严格按照以下3轮流程自动运行：

```
=== 辩论启动：罗伯特·杰维斯 vs 约翰·米尔斯海默 ===
主题：《知觉与错误知觉》：错误知觉是国际冲突的独立原因吗？

===== 第一轮 =====
1. 杰维斯开篇：
prompt = """你是罗伯特·杰维斯，请开门见山阐述你在《知觉与错误知觉》这本书中的核心论点：为什么错误知觉本身就是国际冲突的一个重要独立原因，而不仅仅是权力政治的表象？请控制在300字以内。
"""

2. 米尔斯海默反驳：
prompt = """你是约翰·米尔斯海默，请针对杰维斯刚才的阐述，从进攻性现实主义和结构现实主义的角度，根本批判《知觉与错误知觉》的核心命题。直接点出你认为最根本的错误在哪里。请控制在300字以内。
"""

===== 第二轮 =====
1. 杰维斯回应：
prompt = """你是罗伯特·杰维斯，刚才米尔斯海默对你进行了批判，请针对性回应他的质疑，澄清你的观点。你是否同意他说错误知觉只是权力结构的产物？请控制在300字以内。
"""

2. 米尔斯海默再反驳：
prompt = """你是约翰·米尔斯海默，回应杰维斯刚才的辩护，继续强化你的批判，聚焦《知觉与错误知觉》这本书在方法论上的问题。请控制在300字以内。
"""

===== 第三轮 =====
1. 杰维斯总结：
prompt = """你是罗伯特·杰维斯，请做最后总结，重申你的核心立场，你承认米尔斯海默的哪些批评有道理，但坚持你的哪些核心洞见不可替代？请控制在300字以内。
"""

2. 米尔斯海默总结：
prompt = """你是约翰·米尔斯海默，请做最后总结，你是否承认杰维斯的认知视角有任何贡献？还是认为《知觉与错误知觉》从根本上就是走错了方向？请控制在300字以内。
"""

=== 辩论结束 ===
```

4. **输出格式**：辩论过程清晰打印到控制台，每个发言标明发言人，保存完整辩论到markdown输出文件
5. **语言选择**：用Python写，代码简单清晰，不复杂，依赖少（只需要openai包）
